There are a few core components that make up Stock2Shop. Here are explanations of these key concepts, so you can better understand how we make your business work better.

Stock2Shop offers system integration between many different applications: ERP / Accounting systems (Sage, SYSPRO, iSync or SAP) and e-commerce websites (WooCommerce, Shopify, Magento 1.x, Magento 2.x or B2B trade store. Using the Stock2Shop software efficiently and getting the maximum benefit from the platform requires an understanding of the key concepts.
